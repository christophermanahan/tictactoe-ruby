# Ruby Tic Tac Toe
### A Command Line Tic Tac Toe Game built with Ruby

## Requirements:
  - Ruby 2.5.1
  - Bundler

### Running Tests and Linting
  - "bundle install" to install gems
  - "bundle exec rpsec" to run tests
  - "bundle exec rubocop" to run linter

### Running the Program
  - "ruby ./lib/app.rb" to run game
